Sandbox library to create and use FTP, SSH and SFTP connections.

To use with Composer, add this to your composer.json:

"repositories": [
    //
    {
        "type": "vcs",
        "url": "https://github.com/graymur/remote"
    }
    //
],
"require": {
    //
    "graymur/remote": "dev-master"
    //
}
